# Joung 2023 — Data Download Instructions

This folder contains data derived from the TF Atlas (Joung et al. 2023,
*Nature* 627, 146–154). The raw files are not included in the repository and
must be downloaded before running the preprocessing notebooks.

---

## Required files

Place all files directly in this folder (`data/real/Joung2023/`), **except**
the GRN parquet which goes in `data/real/Joung2023/GRN/` (create the
subfolder if it does not exist).

| File | Source | Used by |
|------|--------|---------|
| `GSE217460_210322_TFAtlas_raw.h5ad` | GEO: GSE217460 | notebook 0 |
| `GSE217460_210322_TFAtlas_differentiated_raw.h5ad` | GEO: GSE217460 | notebook 0 |
| `GSE217066_210715_combinatorial_subsample.h5ad` | GEO: GSE217066 | notebook 0 |
| `GRN/hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet` | CellOracle (see below) | notebook 1 |

---

## Step 1 — Download the single-cell h5ad files from GEO

The three `.h5ad` files are hosted on NCBI GEO. You can download them via the
web browser or with `wget`/`curl` from the terminal.

**GEO accession GSE217460** (main atlas + differentiated subset):

```bash
# Main TF Atlas (~large file, check GEO for current URL)
wget -P data/real/Joung2023/ \
  "https://ftp.ncbi.nlm.nih.gov/geo/series/GSE217nnn/GSE217460/suppl/GSE217460_210322_TFAtlas_raw.h5ad.gz"

# Differentiated subset
wget -P data/real/Joung2023/ \
  "https://ftp.ncbi.nlm.nih.gov/geo/series/GSE217nnn/GSE217460/suppl/GSE217460_210322_TFAtlas_differentiated_raw.h5ad.gz"
```

**GEO accession GSE217066** (combinatorial perturbation subset):

```bash
wget -P data/real/Joung2023/ \
  "https://ftp.ncbi.nlm.nih.gov/geo/series/GSE217nnn/GSE217066/suppl/GSE217066_210715_combinatorial_subsample.h5ad.gz"
```

If the files are compressed (`.gz`), decompress them before use:

```bash
gunzip data/real/Joung2023/*.h5ad.gz
```

> **Tip:** You can also browse and download the files manually from the GEO
> pages:
> - https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE217460
> - https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE217066

## Step 2 — Obtain the CellOracle base GRN (TF motif parquet)

Notebook 1 (`1_GRN_calculation.ipynb`) requires a pre-built TF-motif database
for *Homo sapiens* (hg38) in parquet format:

```
GRN/hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet
```

This file is distributed with CellOracle. The recommended way to obtain it is
to follow the **CellOracle base GRN tutorial** for human data:

> https://morris-lab.github.io/CellOracle.documentation/tutorials/base_grn.html

The tutorial walks through scanning the genome with GimmeMotifs and produces
the parquet file used here. Alternatively, CellOracle provides a convenience
function that may cover your genome:

```python
import celloracle as co
# Check available genomes
co.data.available_base_GRN()
# Load the human promoter base GRN directly (if available for your version)
base_GRN = co.data.load_human_promoter_base_GRN()
```

If `load_human_promoter_base_GRN()` is available for your CellOracle version,
save the result as the parquet file expected by notebook 1:

```python
base_GRN.to_parquet(
    "data/real/Joung2023/GRN/"
    "hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet"
)
```

> **Note:** CellOracle must be run inside the Docker environment described in
> `CELLORACLE_SETUP.md` (at the repository root), as it has conflicting
> dependencies with the main `deep_learning` conda environment.
