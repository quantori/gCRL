# Norman 2019 — Data Download Instructions

This folder contains data from a large-scale Perturb-seq experiment that profiles
gene expression changes in leukemia (K562) cells under genetic perturbations
(Norman et al. 2019, *Science* 365, 786–793).

We use the preprocessed version provided by CPA
(Lotfollahi et al. 2023, *Nature Methods* 20, 1352–1366;
[github.com/facebookresearch/CPA](https://github.com/facebookresearch/CPA)),
following the preprocessing pipeline from our previous work
[SENA](https://github.com/ML4BM-Lab/SENA).

The raw data files are not included in the repository and must be
downloaded before running the preprocessing notebooks.

---

## Required files

Place all files directly in this folder (`data/real/Norman2019/`), **except**
the GRN parquet which goes in `data/real/Norman2019/GRN/` (create the
subfolder if it does not exist).

| File | Source | Used by |
|------|--------|---------|
| `Norman2019_raw.h5ad` | CPA binaries or zip file (see below) | notebooks 1, 2, 3 |
| `GRN/hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet` | CellOracle (shared with Joung pipeline) | notebook 1 |

---

## Step 1 — Obtain `Norman2019_raw.h5ad`

Download and extract the CPA binary archive (~15 GB), copy the required file, then delete the rest:

```bash
# Run from the repository root
wget https://dl.fbaipublicfiles.com/dlp/cpa_binaries.tar
tar -xf cpa_binaries.tar
find . -name "Norman2019_raw.h5ad" -exec cp {} data/real/Norman2019/ \;
rm -rf cpa_binaries.tar pretrained_models/
```

---

## Step 2 — Obtain the CellOracle base GRN (TF motif parquet)

Notebook 1 (`1_GRN_calculation.ipynb`) requires the same pre-built TF-motif
database used by the Joung 2023 pipeline:

```
GRN/hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet
```

If you have already run the Joung 2023 pipeline, you can reuse the file:

```bash
mkdir -p data/real/Norman2019/GRN
cp data/real/Joung2023/GRN/hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet \
   data/real/Norman2019/GRN/
```

Otherwise, follow the instructions in
[`data/real/Joung2023/README.md`](../Joung2023/README.md) (Step 2) to obtain
the file, then place a copy here.

> **Note:** CellOracle must be run inside the Docker environment described in
> the repository README, as it has conflicting dependencies with the main
> `gcrl` conda environment.

---

## Expected folder structure after download

```
data/real/Norman2019/
├── Norman2019_raw.h5ad          # from Option A or B above
└── GRN/
    └── hg38_TFinfo_dataframe_gimmemotifsv5_fpr2_threshold_10_20210630.parquet
```

All other files in this folder (`Norman2019_processed_*.h5ad`,
`Norman2019_pseudobulk_*.h5ad`, `GRN/raw_GRN.csv`, etc.) are generated by
running the preprocessing notebooks in order (1 → 2 → 3).
