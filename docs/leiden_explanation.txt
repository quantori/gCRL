Multiplex Leiden Clustering for TF–TF Regulatory Networks – Seeder Notes
=======================================================================

Goal:
-----
Identify stable, biologically meaningful TF communities by combining:
1. A TF–TF regulatory layer (g_reg)
2. A TF–TF co‑target similarity layer (g_cot)
using multiplex Leiden clustering.

Core Pipeline:
--------------
1. Build g_reg:
   - Undirected TF–TF regulatory graph from GRN dataframe (source, target, coef_abs).
   - Use RBConfigurationVertexPartition (modularity-like).

2. Build g_cot:
   - Compute TF × TG incidence matrix using absolute coefficients.
   - Cosine similarity between TF rows.
   - Apply min_similarity threshold (~0.15).
   - Use RBConfigurationVertexPartition (works better than CPM for this dataset).

3. Multiplex Leiden:
   - One partition per layer:
       * RBConfigurationVertexPartition(g_reg, gamma_reg)
       * RBConfigurationVertexPartition(g_cot, gamma_cot)
   - Use optimise_partition_multiplex with layer_weights=[1.0, cot_weight].
   - No interslice_weight (not supported).
   - Hyperparameters:
        gamma_reg_list = [1, 1.1, 1.2]
        gamma_cot_list = [1, 1.1, 1.2]
        cot_weight_list = [0.1, 0.5, 1.0]

4. Stability assessment:
   - Run each hyperparameter triple over n_seeds=100 seeds.
   - Compute pairwise VI and optional ARI.
   - Sample pairwise comparisons for speed (max_pairs).
   - Skip trivial partitions with too many clusters or mean cluster size < ~3.

5. Consensus clustering:
   - For the selected hyperparameter triple, build co-association matrix from 100 runs.
   - Threshold (e.g., min_coassoc=0.5–0.6).
   - Run CPM or RB again on the co-association graph for final consensus modules.

Key Findings:
-------------
- Using RB for both layers gives good, interpretable structure.
- A min_similarity ≈0.15 for g_cot avoids single giant module.
- Best performing combination in tests:
     (gamma_reg=1.2, gamma_cot=1.0, cot_weight=0.1)
- Resulting modules:
     ~5 major TF communities (sizes 20–34 TFs)
     plus a few small outlier communities.
- Communities are dense in co-target layer and show weaker but non-trivial regulatory connectivity.
- These modules are suitable for biological interpretation (motif/GO enrichment, TF family patterns).

Practical Notes:
----------------
- CPM tends to oversplit dense similarity layers; RB works better here.
- For larger modules, decrease gamma_reg and/or gamma_cot, or reduce cot_weight.
- Building a k-NN version of g_cot (optional) can further stabilize community structure.
- Saving results:
     joblib.dump(results, "multiplex_results.joblib")
     joblib.dump(stability, "multiplex_stability.joblib")
