gcrl.grn.eigengenes — Detailed Behaviour Summary
=================================================

Function: compute_eigengenes
----------------------------
Computes per-community transcription factor (TF) eigengenes as PC1 scores from an AnnData object.
PC1s are fit on a reference subset of cells (typically unperturbed), and all cells are projected
onto these reference-fitted directions. Results are stored directly into the AnnData object.

Key Behaviour
--------------
1. Converts adata.X to dense float32.
2. Uses adata.var['kind'] ∈ {"TF", "TG"}; only "TF" genes contribute.
3. Uses adata.var[community_col] (categorical) to define gene communities.
4. Fits PCA(1) for each community on reference cells (default query: intervention == "unperturbed").
5. Projects all cells using reference scaling and loadings.
6. Also computes a pooled TF eigengene using all TFs.
7. Supports "global" and "by_cell_type" modes.
8. Writes results in-place into adata.obsm and adata.uns.

Inputs
-------
adata: AnnData
community_col: str = "community"
reference_query: str = 'intervention == "unperturbed"'
mode: str = "by_cell_type"   # {"global", "by_cell_type"}
cell_type_col: str = "cell_type"
seed: int = 42

Outputs (in-place)
-------------------
adata.obsm["X_comm_eig"]: (n_cells, n_communities + 1) eigengene matrix
adata.uns["X_comm_eig_comm_ids"]: list of community identifiers
adata.uns["X_comm_eig_global_index"]: index of the pooled TF column
adata.uns["comm_eig_meta"]: metadata dict describing per-community stats and parameters

Computation Modes
------------------
**1. Global Mode ("global")**
   - All cells in the dataset are considered together, regardless of their cell type.
   - A single PCA model (PC1) is fitted per TF-community using reference cells defined by the
     `reference_query` (e.g., unperturbed cells).
   - The resulting eigengenes describe global regulatory module activity across all cells.
   - Use when the GRN structure is shared across all cell types, or when you want a single
     representation of regulatory module activity across the dataset.

**2. By Cell Type Mode ("by_cell_type")**
   - Cells are divided by their `cell_type_col` category (e.g., 'cell_type').
   - Eigengenes are computed independently within each cell type:
        • For each cell type, the same communities (gene sets) are used.
        • PCA is fitted only on reference cells *within that cell type*.
        • The fitted model is applied only to cells of that cell type.
   - The final `X_comm_eig` matrix contains all cells, with eigengene values filled according to
     their respective cell type computations.
   - This mode preserves cell-type-specific scaling and activity directions while maintaining
     consistent column ordering across types.
   - Use when regulatory activity patterns differ substantially between cell types, or when
     evaluating perturbation effects within homogeneous cell populations.

Edge Cases
-----------
- Empty TF set → zero eigengene vector.
- Single-gene TF set → z-scored expression of that gene.
- Missing reference cells → all cells used for fitting.
- Mismatched community categories across cell types → ValueError.
- Missing required columns ('kind', community_col, cell_type_col) → ValueError.

Computation Details
--------------------
For each community:
  - Subset to TF genes.
  - Compute mean/std on reference cells.
  - If >=2 genes: fit PCA(1), project all cells.
  - If 1 gene: z-score expression.
  - If 0 genes: zeros.
Compute pooled eigengene using all TFs similarly.

Metadata
---------
adata.uns["comm_eig_meta"]["per_comm_stats"]: per-community scaling and PCA loadings.
adata.uns["comm_eig_meta"]["per_comm_genes"]: list of TFs used in each community.
adata.uns["comm_eig_meta"]["pooled_tf_stats"]: scaling and loadings for pooled eigengene.

Typical Usage
--------------
from gcrl.grn.eigengenes import compute_eigengenes

compute_eigengenes(
    adata,
    community_col="community",
    reference_query='intervention == "unperturbed"',
    mode="by_cell_type",
    cell_type_col="cell_type",
    seed=42,
)

# Access results
adata.obsm["X_comm_eig"]
adata.uns["X_comm_eig_comm_ids"]
adata.uns["X_comm_eig_global_index"]
adata.uns["comm_eig_meta"]
